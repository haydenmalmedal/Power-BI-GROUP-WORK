# Pokémon Data Dashboard (Power BI)
-GROUP WORK-

## Overview
This project involves building an interactive Power BI dashboard to analyze Pokémon data and uncover patterns in attributes, types, and performance metrics. The goal was to transform data into a visually engaging dashboard that allows users to explore trends and insights.

## Objective
The goal of this project was to:
- Clean and prepare a dataset for visualization
- Build an interactive dashboard in Power BI
- Identify trends across Pokémon attributes (e.g., stats, types)

## Tools & Technologies
- Power BI
- Data Cleaning & Transformation (Power Query)
- Data Visualization & Dashboard Design

## Data
The dataset includes Pokémon characteristics such as:
- Name
- Type (Primary & Secondary)
- Stats (HP, Attack, Defense, Speed, etc.)
- Generation
- Legendary status

## Dashboard Features
- Interactive filters by Pokémon type, generation, and category
- Visual comparisons of key stats (Attack, Defense, Speed, etc.)
- Distribution analysis across different Pokémon types
- Dynamic visuals that update based on user selections

## Analysis
- Cleaned and structured the dataset using Power Query
- Created calculated fields and measures for analysis
- Designed visualizations to highlight key comparisons and trends
- Built an interactive dashboard to improve usability and insights

## Key Insights
- Certain Pokémon types consistently show higher performance in specific stats (e.g., Attack vs. Defense)
- Legendary Pokémon tend to have significantly higher overall stats
- Stat distributions vary widely across generations
- Some types are more balanced, while others are specialized
